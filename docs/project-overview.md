# Project Overview

## Purpose

Project Atlas documents the restoration and modernization of a Dell Latitude E7250 into a reliable Ubuntu infrastructure server.

Rather than purchasing new enterprise hardware, this project demonstrates how an older business-class laptop can be repurposed into a dependable server platform through careful hardware upgrades, physical restoration, and thorough validation.

---

# Objectives

The objectives of Project Atlas are to:

- Extend the useful life of existing hardware.
- Improve performance through hardware upgrades.
- Restore the physical condition of the laptop.
- Improve cooling and long-term reliability.
- Create a stable command-line server platform.
- Document every stage of the project for future reference.

---

# Scope

This repository focuses on:

- Hardware restoration
- SSD upgrade
- Memory upgrade
- Cooling improvements
- Physical repairs
- Local maintenance console
- Hardware validation
- Performance testing

Software services hosted by the server are documented separately within the main homelab repository.

---

# Project Philosophy

Every modification made during Project Atlas will be:

- Planned
- Documented
- Tested
- Photographed where appropriate
- Verified before moving to the next stage

The goal is to produce documentation that another person could follow to recreate the same project.

---

# Current Project Status

The project is currently in the planning and documentation phase.

Hardware upgrades and restoration will be completed in stages, with each milestone recorded in the changelog and supporting documentation.
